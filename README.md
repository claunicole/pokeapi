# Installation

- Ruby version: 3.2.2
- Rails version: 7.1.1

- Database: SQLite


Up and Running:
1. Clone this repository 
2. Run the command `bundle install`
3. Create the database with `rails db:create`
4. Run the migrations `rails db:migrate`
5. Seed the dabase `rake import:import_pokemon`
